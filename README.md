# 🎢 Adventure Park Landing Page

A visually stunning, responsive landing page for a modern Adventure Theme Park. This project showcases the use of **Bootstrap 5** for layout and **Custom CSS** for high-quality UI/UX design.

## ✨ Features
- **Responsive Navigation**: Smooth sticky navbar for easy access.
- **Hero Section**: An immersive full-screen hero banner with a gradient overlay.
- **Activity Showcase**: Card-based layout displaying various park attractions like Water Slides, Wave Pools, and Rollercoasters.
- **Mobile Friendly**: Fully responsive design that works on desktops, tablets, and smartphones.
- **Bootstrap 5 Integration**: Utilizes the latest Bootstrap grid system and components.

## 🛠️ Tech Stack
- **HTML5**: For semantic web structure.
- **CSS3**: For custom styling and background effects.
- **Bootstrap 5.3.8**: For rapid responsive development and pre-styled components.
